# DDLKiller
<游戏介绍>
DDLKiller是一款全自制VR射击游戏(不是很精美)。校园中，DDL怪正在侵蚀你的生活 现在开始消灭它！小心不要被他吞噬，变成拖延患者哦。
<玩法介绍>
校园中会随机出现多种DDL怪，瞄准射击DDL怪。你只有8滴「马上就做」血，小心不要被怪物吞噬。60s内结束游戏，结算分数，你会是那个「当代卷王」吗?
